Bootstrap divides up the page into 12 columns. Have a div with class 'row', inside it have sub-divs each of them with class 'col'. Bootstrap divides up the 12 columns among the number of columns you have equally. For instance, if you have only 3 sub-divs of class 'column', each one will take 4 columns; 1/3 of the page, which makes it realtive and fluid.
